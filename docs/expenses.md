# 経費まとめ

このブースで発生する経費を、**経費計上しやすい形**で一覧化します。金額は税込の目安で、**発注・購入の確定額で更新**してください。数量根拠は [造形プリント環境と必要量](/printing) を参照。

::: tip 経理へ渡すときのポイント
- **本体（3Dプリンター）は10万円以上**のため、原則は消耗品ではなく**固定資産（工具器具備品）**として計上し減価償却の対象になります。少額減価償却資産の特例（30万円未満・年間合計300万円まで即時償却）や一括償却資産（20万円未満）の適用可否は**経理／税理士に確認**してください。
- フィラメント・撮影ブース資材・什器などは**消耗品費**として当期経費に計上できます。
- 用途はすべて「**AI Dev Day 2026 スタックチャンブース**（2026-07-24）」に紐付きます。領収書・納品書はこのイベント名で保管。
:::
