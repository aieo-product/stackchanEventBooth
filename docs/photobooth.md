# 簡易撮影ブース

着せ替えたスタックチャンを置いて記念撮影できる、卓上サイズの撮影ブースです。「**Happy Birthday スタックチャン**」のお誕生日パーティ ジオラマがコンセプト。

## コンセプト

スマホで撮るだけでSNS映えする画づくりを、卓上の小さなステージで実現します。来場者は着せ替えたスタックチャンを定位置に置いて、その場で撮影 → `#ｽﾀｯｸﾁｬﾝお誕生日2026` で投稿、という体験を楽しめます。

## 構成要素

| 要素 | 内容 |
|---|---|
| 🪧 **背景ボード** | 卓上・自立式（W600〜900 × H400〜600 目安）。折りたたみ収納可。木目調＋パステルの壁紙 |
| 🎉 **ガーランド／バナー** | "HAPPY BIRTHDAY STACK-CHAN" の文字飾り・風船モチーフ |
| 🪑 **ミニ家具・小物** | 椅子・テーブル・ケーキ・バルーン文字など（スタックチャンサイズのジオラマ小物） |
| 📍 **定位置マーカー** | スタックチャンを置く撮影位置の目印 |
| 💡 **卓上照明** | 安定した明るさのLED。推奨アングル・ハッシュタグ掲示を併設 |

## 撮影の流れ

1. 好きなアクセサリーでスタックチャンを着せ替える
2. 撮影ブースの定位置にスタックチャンを置く
3. スマホで撮影（推奨アングルを掲示）
4. `#ｽﾀｯｸﾁｬﾝお誕生日2026` で投稿

## 設営の要件

- 卓上 **W900 × D450** 程度に収まる設計（展示テーブルの一角を使用）
- 自立式・折りたたみ式で、**設営／撤収は5分以内**
- 搬入しやすい軽量・コンパクト構成

> ベースのアイデアは、コミュニティのお誕生日会で見られた卓上ジオラマ撮影スポットです。撮影ブースの装飾・小物は本企画で新規制作します（他者の造形を流用する場合は許諾を取得）。
