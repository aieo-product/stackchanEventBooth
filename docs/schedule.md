# 当日運営（13:00–17:00）

会場は **1F コミュニティショーケース**。**「コミュニティが1時間シフトで交代」** する運用のため、**1枠＝1時間** を単位に、素早く設営・撤収できる構成にします。

::: warning ⏱ 出展枠は要確認
「1時間枠×複数回」「1回のみ」「13–17通し」のいずれかは主催に確認中です。下記は **1時間枠を複数回担当する場合** を想定したタイムラインです（→ [必要スペース](/space) 参照）。
:::

## タイムライン（1時間枠を想定）

| タイミング | 内容 | 目安時間 |
|---|---|---|
| 枠の開始前 | **搬入・設営**（テーブルにクロス→アクセサリー陳列→撮影ブース展開→実機起動） | **5〜10分** |
| 枠の最初 | 最終チェック（陳列・撮影ブース・掲示物・消毒） | 〜2分 |
| 枠の本番 | 運営（着せ替え対応・撮影サポート・補充・実機メンテ） | 約50分 |
| 枠の終了前 | **撤収**（小物をコンテナへ→次コミュニティへ引き継ぎ） | **5分** |

### 通し（13–17）想定の場合のタイムライン

| 時刻 | 内容 |
|---|---|
| 〜12:30 | 搬入・設営 |
| **13:00** | ブースオープン 🎉 |
| 13:00–17:00 | 運営（適宜アクセサリー補充・実機メンテ）|
| **17:00** | クローズ・撤収開始 |

::: tip 交代運用への対応
- **卓上完結・モジュール化**：アクセサリーはトレイ単位、撮影ブースは折りたたみ式で、**5〜10分で設営/撤収**。
- 折りたたみコンテナ1〜2個に全什器を収納し、交代をスムーズに。
:::

## 来場者の体験フロー

1. 🎀 好きなアクセサリーでスタックチャンを着せ替え
2. 📸 撮影ブースで記念撮影
3. 📲 `#ｽﾀｯｸﾁｬﾝお誕生日2026` で投稿
4. 🎁 （任意）ノベルティ（キーホルダー／ステッカー）

## 運営体制（ドラフト）

- スタッフ **2名以上** を推奨（着せ替え案内係／撮影・投稿サポート係）
- 共用アクセサリーの **衛生対策**（消毒・クリーニング）を運用に組み込み
- 実機スタックチャンは **着せ替え用／撮影用** で複数台用意できると回転が良い
