# 当日運営（13:00–17:00）

ブースの運営時間は **13:00–17:00 の4時間** を想定しています。

## タイムライン

| 時刻 | 内容 |
|---|---|
| 〜12:30 | **搬入・設営**（テーブル設置、展示・撮影ブース準備、実機起動確認） |
| 12:30–13:00 | 最終チェック（アクセサリー陳列・撮影ブース・掲示物） |
| **13:00** | **ブースオープン** 🎉 |
| 13:00–17:00 | 運営（着せ替え対応・撮影サポート・アクセサリー補充・実機メンテ） |
| **17:00** | **クローズ・撤収開始** |

## 来場者の体験フロー

1. 🎀 好きなアクセサリーでスタックチャンを着せ替え
2. 📸 撮影ブースで記念撮影
3. 📲 `#ｽﾀｯｸﾁｬﾝお誕生日2026` で投稿
4. 🎁 （任意）ノベルティ（キーホルダー／ステッカー）

## 運営体制（ドラフト）

- スタッフ **2名以上** を推奨（着せ替え案内係／撮影・投稿サポート係）
- 共用アクセサリーの **衛生対策**（消毒・クリーニング）を運用に組み込み
- 実機スタックチャンは **着せ替え用／撮影用** で複数台用意できると回転が良い
