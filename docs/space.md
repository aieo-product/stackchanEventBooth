# 必要スペース

ブースに必要な広さと、その中のレイアウトをまとめます。会場レギュレーション確認後に確定しますが、**最小 2.0m × 2.0m の1区画** があれば成立する想定です。

## 結論：どれくらいの広さが必要か

| パターン | 広さ | 内容 | 適する場合 |
|---|---|---|---|
| **最小構成** | **2.0m × 2.0m**（4㎡） | 展示テーブル1台＋撮影ブース＋来場者2〜3名の回遊 | 標準的な1ブース区画 |
| **推奨構成** | **2.0m × 3.0m**（6㎡） | 上記＋待機列・スタッフ動線に余裕 | 混雑が見込まれる場合 |
| **省スペース** | 1.8m × 1.8m | テーブルを撮影ブースと兼用、回遊を縮小 | スペースが限られる場合 |

::: tip ポイント
撮影ブースは **卓上（W900×D450）** に収まるので、専用の床面積は不要です。展示テーブルの一角に置けるため、**実質はテーブル1台分＋来場者が立つスペース** があれば運営できます。
:::

### ゾーンの内訳

| ゾーン | サイズ目安 | 内容 |
|---|---|---|
| 🟨 展示テーブル | W1800 × D600mm | 着せ替えアクセサリーを並べる。スタッフはテーブル内側に立つ |
| 🟥 撮影ブース | 卓上 W900 × D450mm | テーブル右側に設置するジオラマ撮影スポット |
| 🟦 回遊スペース | 約 2.0m × 0.9m | 来場者が着せ替え・撮影・投稿を行う動線（2〜3名同時） |

## 設備・什器の要件

| 項目 | 要件 | 備考 |
|---|---|---|
| テーブル | W1800×D600 ×1（会場貸出 or 持込） | クロス掛け |
| 椅子 | スタッフ用 1〜2 | — |
| 電源 | 1口〜（要確認） | 実機スタックチャン・卓上照明用。モバイルバッテリーで代替可 |
| 照明 | 卓上LED ×1 | 撮影品質の安定用 |
| Wi-Fi | 任意 | 実機のデモ内容による |
| 搬入 | 折りたたみ什器で台車1台程度 | 撮影ブースは折りたたみ収納 |

::: warning 確定前の確認事項
- 会場から割り当てられるブース寸法・テーブル有無
- 電源の提供可否・容量
- 搬入経路／搬入出時間
:::
