# stackchanEventBooth

**AI Dev Day 2026（2026年7月24日・大手町プレイスホール＆カンファレンス）** のスタックチャンブース企画・運営リポジトリ。

イベントを盛り上げるための **スタックチャン用アクセサリー** と **簡易撮影ブース** を制作し、来場者に「スタックチャンを着せ替えて撮って楽しむ」体験を提供します。

📄 **公開提案サイト（Cloudflare Pages）: https://stackchan-event-booth.pages.dev/**
🎤 LTスライド（リッチHTML）: https://stackchan-event-booth.pages.dev/slides/lt.html

## このリポジトリの目的

- ブース企画（アクセサリー／撮影ブース／必要スペース）の管理
- アクセサリー造形例の調査と、各制作者への **許諾取得状況** の管理
- 関係者向け公開提案サイト（Cloudflare Pages）のソース管理

## ブース概要（想定）

| 項目 | 内容 |
|---|---|
| イベント | AI Dev Day 2026 |
| 日時 | 2026年7月24日（金）|
| ブース運営時間 | **13:00–17:00**（4時間想定）|
| 会場 | 大手町プレイスホール＆カンファレンス（東京・大手町）|
| 体験 | スタックチャンの着せ替えアクセサリー展示＋記念撮影ブース |

## アクセサリーの取り扱い方針

本ブースで展示・配布・印刷するアクセサリーは **既存造形のコピーではなく、各制作者に許諾を得たうえで設置** します。調査した造形例と制作者アカウントは [`research/accessories.md`](research/accessories.md) で管理し、許諾ステータスを記録します。
